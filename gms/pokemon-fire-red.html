Play Pokemon Fire Red

WARNING: The emulator saves the progress to the cache, so it's advised to use the Save State option whenever you want to save your game. If the cache is cleaned all progress will be lost

Description:

Pokemon Fire Red is a role-playing game where players embark on a journey to become a Pokemon Trainer Champion. The objective is to capture and train Pokemon creatures, battle other trainers, and collect Gym Badges to ultimately challenge the Elite Four and the Champion.

Controls:

Move with the arrow keys

Z -> Interact/Select an option

X -> Cancel/Run (You need the running shoes to run)

Enter -> Menu

Shift -> Use a registered item

A/S -> Help Menu
