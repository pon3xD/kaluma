Play Grand Theft Auto Advance

WARNING: The emulator saves the progress to the cache, so it's advised to use the Save State option whenever you want to save your game. If the cache is cleaned all progress will be lost

Description:

Play the original Grand Theft Auto for the Gameboy Advance directly in this site! Complete missions, steal cars and do whatever you want in this old-school criminal sandbox game.

Controls:

On foot:

Move with the arrow keys

Z -> Attack

X -> Sprint

Z+X -> Jump

A -> Enter Vehicle

Shift -> Select Weapon

Enter -> Pause

In Vehicle:

Z -> Accelerate

X -> Brake/Reverse

Steer with left/right arrow keys

A -> Exit vehicle

Z+X -> Turret Fire

Arrow key+S -> Horn/Siren

A+S -> Sub-mission
