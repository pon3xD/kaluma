Play Pokemon Platinum

WARNING: The emulator saves the progress to the cache, so it's advised to use the Save State option whenever you want to save your game. If the cache is cleaned all progress will be lost

Description:

Pokemon Platinum brings the adventure you're familiar with to the DS, with more Pokemons, Gyms, Adventures and Secrets. Find out teams galactics' plan and challenge the elite four to become the Ultimate Pokemon Trainer!

Controls:

Move with the arrows

Z -> 'A' button (Interact)

X -> 'B' button (Cancel)

A -> 'Y' button (Registered Items)

Z -> 'X' button (Menu)

Q -> 'L' button

W -> 'R' button

Enter -> Start

Shift -> Select
