Play Street Fighter 3: Third Strike

WARNING: The emulator saves the progress to the cache, so it's advised to use the Save State option whenever you want to save your game. If the cache is cleaned all progress will be lost

Description:

Street Fighter 3: Third Strike is a classic fighting game with solid mechanics and gameplay, making it one of the best fighting games of all time. The objective is simple, defeat every adversary until you reach the new generation of god, Gill.

Controls:

Move with the arrows

A -> Light Punch

S -> Medium Punch

D -> Heavy Punch

Z -> Light Kick

X -> Medium Kick

C -> Heavy Kick

Enter -> Start

Shift -> Coin

ProTip: You need to insert at least 2 coins to start playing
